Welcome to BlogHub

This is our first blog post! BlogHub is a GitHub-powered blogging platform that makes it easy to publish articles.

Features

  • Write in Markdown using GitHub Issues
  • Automatic HTML generation
  • Beautiful, responsive design
  • GitHub Pages hosting

How to Create Posts

  1. Create a new GitHub Issue
  2. Write your content in Markdown
  3. Add the APPROVED label
  4. Close the issue to publish

That's it! Your post will be automatically generated and published.
